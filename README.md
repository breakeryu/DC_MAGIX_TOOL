# DC_MAGIX_TOOL
这是一个电路计算工具

# 1.DC参数计算工具介绍
在进行DC直流电压输出时，通常都需要计算如下图所示分压电阻R1,R2和输出电压Vout参数。这款使用EXCEL制作的小工具可以帮助我们来方便的计算这些参数。
![](https://img2020.cnblogs.com/blog/1423856/202201/1423856-20220108161807324-2087648626.png)
